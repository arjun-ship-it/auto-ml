# Feature Specs Index

This directory contains feature specifications for the AutoML Agent project.

## Workflow

1. **New Feature**: Run `/feature-new` to create branch + spec
2. **Plan**: Run `/plan <slug>` to fill out the spec
3. **Execute**: Implement according to spec
4. **PR**: Run `/pr <slug>` to create pull request
5. **Review**: Address Gemini Code Assist comments
6. **Submit**: Run `/submit <slug>` to merge

## Status Legend

- **Draft**: Spec created, not yet planned
- **Planned**: Spec complete, ready for implementation
- **In Progress**: Implementation underway
- **In Review**: PR created, awaiting review
- **Done**: Merged to main
